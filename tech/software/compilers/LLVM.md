## FORTRAN

- Official subproject of LLVM in the LLVM GitHub repo.
- MLIR-based.

Features:
- Fortran 2023 support.
- CPU & GPU Fortran Standard Parallelism (do concurrent loops, array intrinsics).
- CPU & GPU OpenACC and OpenMP
- support.
- CUDA Fortran Support.
