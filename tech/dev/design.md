## Help

### Command Line

[docopt](http://docopt.org/): Pythonic command-line interface description language - deprecated, but useful as reference for layout
Example:

```
Naval Fate.

Usage:
  naval_fate -h | --help
  naval_fate --version

Options:
  -h --help     Show this screen.
  --version     Show version.
```
